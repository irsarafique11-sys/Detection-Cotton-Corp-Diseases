# Cotton Plant Disease Detector

A professional Android application designed to detect diseases in cotton plants using cutting-edge Machine Learning. This app empowers farmers with real-time diagnostics to ensure healthy crops and sustainable yields.

## üåü Key Features
- **Real-time Detection**: Capture images directly using the device camera for instant analysis.
- **Gallery Support**: Upload existing photos from your gallery for disease identification.
- **Multi-lingual Results**: Disease information provided in English, Urdu, and Hindi for better accessibility.
- **Deep Learning Core**: Powered by a Convolutional Neural Network (CNN) model optimized for mobile performance via TensorFlow Lite.

## üöÄ Getting Started
### Prerequisites
- Android Studio Ladybug or newer
- JDK 17+ (JDK 25 recommended)
- Android Device or Emulator (API 26+)

### Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/CottonPlantDiseaseDetection.git
   ```
2. Open the project in **Android Studio**.
3. Sync Project with Gradle Files.
4. Click **Run** to install on your device.
